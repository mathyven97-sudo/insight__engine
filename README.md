# Social Media Insight Engine

## How to Run Locally

1. **Prerequisites**: Ensure you have Python installed.
2. **Setup**: The project uses a virtual environment in `backend/venv`.
3. **Run**:
   - Double-click `run_server.bat` in this directory.
   - Or manually:
     ```bash
     cd backend
     .\venv\Scripts\activate
     uvicorn app.main:app --reload
     ```
4. **Access**: Open [http://localhost:8000](http://localhost:8000) in your browser.
